Sherit Windows 版 v1.0.0-1

【推荐启动方式】
1. 首次使用：双击「安装运行库.bat」（只需安装一次）
2. 之后每次：双击「启动 Sherit.bat」

【安装步骤】
1. 右键 zip -> 全部解压，解压到一个文件夹（如 D:\Sherit）
2. 双击「安装运行库.bat」，等待安装完成
3. 双击「启动 Sherit.bat」启动应用

【若仍提示缺少 flutter_windows.dll】
1. 运行「检查安装.bat」，看哪些文件显示 [缺少]
2. 关闭杀毒软件后重新解压
3. 右键 zip -> 属性 -> 勾选「解除锁定」-> 确定，再解压
4. 不要从压缩包内直接运行，必须完整解压

【从 GitHub Actions 下载时】
下载的 artifact 解压后如果还有一个 zip，需要再解压一次

【若提示 VCRUNTIME140_1.dll 缺失】
双击「安装运行库.bat」，或手动下载:
https://aka.ms/vs/17/release/vc_redist.x64.exe

【系统要求】
- Windows 10/11 64 位（不支持 32 位系统）

包信息: exe=1.31MB, 总计=32.81MB
